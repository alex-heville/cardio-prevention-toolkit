# Initial Toolkit Release (v1.0)

## Summary
This is the first version of the Cardiovascular Prevention Toolkit. It includes an evidence-based patient handout comparing lifestyle changes versus statin therapy for cardiovascular risk reduction.

## Contents
- `lifestyle-vs-statin-handout.md`: A one-page patient handout that provides a balanced comparison between starting with lifestyle modifications (diet, exercise) versus starting with a statin for patients with moderate cardiovascular risk. The handout is written in a cautious, evidence-based tone suitable for patient education.

## Status
- Research components (Issues #1 and #2) are still in progress.
- This release represents the first collaborative output of the toolkit team.
